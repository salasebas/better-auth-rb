# openauth-telemetry

Alias package for `better_auth-telemetry`.

## Installation

```ruby
gem "openauth-telemetry"
```

```ruby
require "openauth/telemetry"
```

This package depends on `better_auth-telemetry` and loads the canonical Better Auth Ruby telemetry package.

For documentation, visit https://better-auth-rb.vercel.app/.
