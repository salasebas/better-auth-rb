## Installation

```ruby title="Gemfile"
gem "better_auth-roda"
```

```bash
bundle install
```

## Setup

<Steps>

### Enable the plugin

```ruby title="app.rb"
require "roda"
require "better_auth/roda"

class App < Roda
  plugin :better_auth

  better_auth at: "/api/auth" do |config|
    config.secret = ENV.fetch("BETTER_AUTH_SECRET")
    config.base_url = ENV["BETTER_AUTH_URL"]
    config.database = :memory  # replace with SQL adapter in production
    config.email_and_password = { enabled: true }
  end

  route do |r|
    r.better_auth

    r.get "dashboard" do
      current_user.to_json
    end
  end
end
```

### Rake tasks

```ruby title="Rakefile"
require "better_auth/roda/tasks"
```

```bash
rake better_auth:install
rake better_auth:generate:migration
rake better_auth:migrate
rake better_auth:doctor
```

</Steps>

Configure the desired core database adapter explicitly — Roda does not pick a default SQL backend.

## Verify

```bash
curl http://localhost:9292/api/auth/ok
```
