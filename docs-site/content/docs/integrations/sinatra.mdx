## Installation

```ruby title="Gemfile"
gem "better_auth-sinatra"
```

```bash
bundle install
```

## Setup

<Steps>

### Register the extension

```ruby title="app.rb"
require "sinatra/base"
require "better_auth/sinatra"

class App < Sinatra::Base
  register BetterAuth::Sinatra

  better_auth at: "/api/auth" do |config|
    config.secret = ENV.fetch("BETTER_AUTH_SECRET")
    config.base_url = ENV.fetch("BETTER_AUTH_URL")
    config.database = ->(options) {
      BetterAuth::Adapters::Postgres.new(options, url: ENV.fetch("DATABASE_URL"))
    }
    config.email_and_password = { enabled: true }
  end

  get "/dashboard" do
    require_authentication
    current_user.fetch("email")
  end
end
```

Call `better_auth` once per Sinatra app class.

### Rake tasks

```ruby title="Rakefile"
require "better_auth/sinatra/tasks"
```

```bash
rake better_auth:install
rake better_auth:generate:migration
rake better_auth:migrate
rake better_auth:doctor
```

</Steps>

## Mount patterns

- **Nested mount:** parent app at `/api`, auth at `better_auth at: "/auth"` → routes at `/api/auth/*`
- **Shared prefix:** Sinatra app mounted at `/api/auth` with `better_auth at: "/api/auth"`

Set `config.base_url` to the public URL users see, including any parent mount prefix.

## Helpers

`current_session`, `current_user`, `authenticated?`, and `require_authentication` delegate to the core session API.

## Verify

```bash
curl http://localhost:9292/api/auth/ok
```
