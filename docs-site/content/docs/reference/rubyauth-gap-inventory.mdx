This holding page lists current under-development or unsupported areas for later triage. It is not linked from the sidebar and does not mark future or unsupported items as supported.

## Real Gaps

| Area | Current RubyAuth status |
| --- | --- |
| [Test Utils](/docs/plugins/test-utils) | RubyAuth does not expose `BetterAuth::Plugins.test_utils` or a public test helper module. |
| [Open API](/docs/plugins/open-api) | The plugin docs currently describe Open API as early-stage and still filling gaps. |
| [Programmatic migrations / `getMigrations`](/docs/concepts/database#programmatic-migrations) | RubyAuth does not expose server-side `getMigrations`; users should use CLI-generated SQL, `bundle exec better-auth migrate`, or framework migrations. |
| [`reference/options` advanced option](/docs/reference/options#advanced) | The upstream parity section is not fully exposed or enforced server-side in Ruby. |
