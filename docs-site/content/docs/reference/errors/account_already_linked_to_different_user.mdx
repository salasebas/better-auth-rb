---
title: Account Already Linked To Different User
description: OAuth error `account_already_linked_to_different_user` in RubyAuth.
---

## When this appears

Provider account is linked to another user in your database.

Redirect URL shape:

```
/api/auth/error?error=account_already_linked_to_different_user
```

Or your configured `errorCallbackURL` with the same query parameters.

## Common causes

- Shared OAuth account across users
- Stale test data

## How to fix

- Unlink from other user via admin tools
- Use separate provider apps for staging

## Diagnostics

```bash
bundle exec better-auth doctor --cwd . --config config/better_auth.rb
```
