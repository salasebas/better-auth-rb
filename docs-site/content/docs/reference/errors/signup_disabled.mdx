---
title: Signup Disabled
description: OAuth error `signup_disabled` in RubyAuth.
---

## When this appears

OAuth attempted to create a new user but sign-up is disabled.

Redirect URL shape:

```
/api/auth/error?error=signup_disabled
```

Or your configured `errorCallbackURL` with the same query parameters.

## Common causes

- `disableSignUp` on provider or global policy
- Invite-only app

## How to fix

- Create user through admin flow first
- Adjust provider `disable_sign_up` options

## Diagnostics

```bash
bundle exec better-auth doctor --cwd . --config config/better_auth.rb
```
