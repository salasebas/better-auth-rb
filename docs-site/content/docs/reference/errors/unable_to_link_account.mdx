---
title: Unable To Link Account
description: OAuth error `unable_to_link_account` in RubyAuth.
---

## When this appears

Account linking is not allowed for this provider or email.

Redirect URL shape:

```
/api/auth/error?error=unable_to_link_account
```

Or your configured `errorCallbackURL` with the same query parameters.

## Common causes

- Implicit sign-up disabled for provider
- Linking rules block provider
- Email domain restricted

## How to fix

- Enable linking in account options
- Use explicit link flow while authenticated

## Diagnostics

```bash
bundle exec better-auth doctor --cwd . --config config/better_auth.rb
```
