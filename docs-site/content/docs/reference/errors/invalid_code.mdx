---
title: Invalid Code
description: OAuth error `invalid_code` in RubyAuth.
---

## When this appears

Authorization code exchange with the provider failed.

Redirect URL shape:

```
/api/auth/error?error=invalid_code
```

Or your configured `errorCallbackURL` with the same query parameters.

## Common causes

- Code already used
- Clock skew
- Wrong client secret
- PKCE verifier mismatch

## How to fix

- Verify `client_id` and `client_secret`
- For Google/Vercel, ensure PKCE flow completes in one browser session

## Diagnostics

```bash
bundle exec better-auth doctor --cwd . --config config/better_auth.rb
```
