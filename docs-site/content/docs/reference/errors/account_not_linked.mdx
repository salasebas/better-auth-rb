---
title: Account Not Linked
description: OAuth error `account_not_linked` in RubyAuth.
---

## When this appears

Existing user tried OAuth sign-in but account is not linked to that provider.

Redirect URL shape:

```
/api/auth/error?error=account_not_linked
```

Or your configured `errorCallbackURL` with the same query parameters.

## Common causes

- User previously signed up with email/password only
- Linking required before OAuth sign-in

## How to fix

- Sign in with email/password then link provider
- Enable implicit linking if appropriate for your app

## Diagnostics

```bash
bundle exec better-auth doctor --cwd . --config config/better_auth.rb
```
