---
title: Email Not Found
description: OAuth error `email_not_found` in RubyAuth.
---

## When this appears

Provider user profile did not include an email address.

Redirect URL shape:

```
/api/auth/error?error=email_not_found
```

Or your configured `errorCallbackURL` with the same query parameters.

## Common causes

- Provider scope missing email
- Account privacy settings
- Enterprise IdP omitting email claim

## How to fix

- Request `email` scope from provider
- Use a provider that returns email for your tenant

## Diagnostics

```bash
bundle exec better-auth doctor --cwd . --config config/better_auth.rb
```
