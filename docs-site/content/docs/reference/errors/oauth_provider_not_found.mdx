---
title: OAuth Provider Not Found
description: OAuth error `oauth_provider_not_found` in RubyAuth.
---

## When this appears

Callback `providerId` is not configured in `social_providers`.

Redirect URL shape:

```
/api/auth/error?error=oauth_provider_not_found
```

Or your configured `errorCallbackURL` with the same query parameters.

## Common causes

- Typo in provider id
- Plugin provider not loaded
- Stale callback URL from old config

## How to fix

- Match callback segment to configured provider id
- Restart app after changing `social_providers`

## Diagnostics

```bash
bundle exec better-auth doctor --cwd . --config config/better_auth.rb
```
