---

## When this appears

Generic OAuth plugin state validation failed or session expired mid-flow.

Redirect URL shape:

```
/api/auth/error?error=please_restart_the_process
```

Or your configured `errorCallbackURL` with the same query parameters.

## Common causes

- Stale generic OAuth state
- Multiple tabs racing OAuth
- Cookie cleared during flow

## How to fix

- Start OAuth from `/api/auth/sign-in/oauth2` again
- Avoid parallel OAuth attempts

## Diagnostics

```bash
bundle exec better-auth doctor --cwd . --config config/better_auth.rb
```
