---
title: Unknown
description: OAuth error `unknown` in RubyAuth.
---

## When this appears

Unhandled OAuth error slug or provider-returned error.

Redirect URL shape:

```
/api/auth/error?error=unknown
```

Or your configured `errorCallbackURL` with the same query parameters.

## Common causes

- Provider sent unexpected error query param
- Custom plugin error

## How to fix

- Check `error_description` query param
- Inspect server logs
- Run `bundle exec better-auth doctor`

## Diagnostics

```bash
bundle exec better-auth doctor --cwd . --config config/better_auth.rb
```
