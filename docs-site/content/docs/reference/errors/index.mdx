Social and generic OAuth flows redirect to `/api/auth/error` (or your `errorCallbackURL`) with an `error` query parameter when sign-in fails.

## RubyAuth redirect errors

* [Account Already Linked To Different User](/docs/reference/errors/account_already_linked_to_different_user)
* [Account Not Linked](/docs/reference/errors/account_not_linked)
* [Email Doesn't Match](/docs/reference/errors/email_doesn't_match)
* [Email Not Found](/docs/reference/errors/email_not_found)
* [Invalid Code](/docs/reference/errors/invalid_code)
* [No Code](/docs/reference/errors/no_code)
* [OAuth Provider Not Found](/docs/reference/errors/oauth_provider_not_found)
* [Please Restart The Process](/docs/reference/errors/please_restart_the_process)
* [Signup Disabled](/docs/reference/errors/signup_disabled)
* [State Mismatch](/docs/reference/errors/state_mismatch)
* [State Not Found](/docs/reference/errors/state_not_found)
* [Unable To Get User Info](/docs/reference/errors/unable_to_get_user_info)
* [Unable To Link Account](/docs/reference/errors/unable_to_link_account)
* [Unknown](/docs/reference/errors/unknown)

## Upstream-only slugs

These slugs exist in upstream Better Auth docs but are **not** emitted by RubyAuth OAuth redirects:

* [Unable To Create User](/docs/reference/errors/unable_to_create_user)
* [Unable To Create Session](/docs/reference/errors/unable_to_create_session)
* [Invalid Callback Request](/docs/reference/errors/invalid_callback_request)
* [No Callback Url](/docs/reference/errors/no_callback_url)
* [Internal Server Error](/docs/reference/errors/internal_server_error)

For API JSON errors, configure [`on_api_error`](/docs/reference/options#on-api-error) and inspect `BetterAuth::APIError` in server logs.
