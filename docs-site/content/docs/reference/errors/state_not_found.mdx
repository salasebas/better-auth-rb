---
title: State Not Found
description: OAuth error `state_not_found` in RubyAuth.
---

## When this appears

OAuth callback arrived without valid stored OAuth state.

Redirect URL shape:

```
/api/auth/error?error=state_not_found
```

Or your configured `errorCallbackURL` with the same query parameters.

## Common causes

- State cookie expired (10 minute default)
- Session cleared mid-flow
- Multiple auth mounts with mismatched cookie paths

## How to fix

- Retry sign-in from the beginning
- Verify single `better_auth` mount per app
- Check cookie `SameSite` and domain settings

## Diagnostics

```bash
bundle exec better-auth doctor --cwd . --config config/better_auth.rb
```
