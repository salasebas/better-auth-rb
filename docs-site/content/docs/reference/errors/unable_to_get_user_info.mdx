---
title: Unable To Get User Info
description: OAuth error `unable_to_get_user_info` in RubyAuth.
---

## When this appears

Token exchange succeeded but user profile could not be fetched.

Redirect URL shape:

```
/api/auth/error?error=unable_to_get_user_info
```

Or your configured `errorCallbackURL` with the same query parameters.

## Common causes

- Revoked tokens
- Provider API outage
- Missing userinfo scope

## How to fix

- Check provider status
- Verify scopes include profile/email
- Inspect server logs for provider HTTP errors

## Diagnostics

```bash
bundle exec better-auth doctor --cwd . --config config/better_auth.rb
```
