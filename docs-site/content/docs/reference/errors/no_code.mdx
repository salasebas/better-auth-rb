---
title: No Code
description: OAuth error `no_code` in RubyAuth.
---

## When this appears

Provider redirected back without an authorization `code`.

Redirect URL shape:

```
/api/auth/error?error=no_code
```

Or your configured `errorCallbackURL` with the same query parameters.

## Common causes

- User denied consent
- Provider misconfiguration
- Wrong redirect URI registered

## How to fix

- Confirm redirect URI matches `{BETTER_AUTH_URL}/api/auth/callback/{provider}`
- Check provider console for errors

## Diagnostics

```bash
bundle exec better-auth doctor --cwd . --config config/better_auth.rb
```
