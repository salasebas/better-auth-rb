---
title: Email Doesn't Match
description: OAuth error `email_doesn't_match` in RubyAuth.
---

## When this appears

Linking a social account but provider email differs from signed-in user.

Redirect URL shape:

```
/api/auth/error?error=email_doesn't_match
```

Or your configured `errorCallbackURL` with the same query parameters.

## Common causes

- User signed in with different email than OAuth profile
- Account linking rules enforced

## How to fix

- Sign in with matching email first
- Review `account.account_linking` options in configuration

## Diagnostics

```bash
bundle exec better-auth doctor --cwd . --config config/better_auth.rb
```
