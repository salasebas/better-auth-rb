RubyAuth is a **server-first** authentication and authorization library for Ruby. It exposes a Rack-compatible auth handler, a programmatic `auth.api` surface, and a plugin ecosystem — so you can add email/password, OAuth, 2FA, organizations, and more without rebuilding auth infrastructure.

The design is inspired by [Better Auth](https://www.better-auth.com), but RubyAuth is an independent Ruby implementation maintained in this repository. The repository pins an upstream release as a reproducible reference; that pin does not imply complete behavioral parity. API shapes follow the Ruby gems here, not the upstream TypeScript docs.

## Features

RubyAuth provides a wide range of **server-side** authentication features for Ruby apps and extends via plugins. Supported capabilities are documented here; features still catching up to the upstream design are marked **Under development**.
